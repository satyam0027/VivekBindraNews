# Week 2 — On-Page SEO Checklist (Per Article)
**Apply this checklist to EVERY article before publishing**

---

## Pre-Publishing Checklist (Run for Each Article)

### ✅ Title & Meta
- [ ] Title tag includes primary keyword near the beginning
- [ ] Title tag is 50–60 characters long
- [ ] Title tag format: `[Keyword]: [Descriptive hook] | Vivek Bindra News`
- [ ] Meta description is 145–160 characters
- [ ] Meta description includes primary keyword once
- [ ] Meta description has a compelling reason to click
- [ ] URL slug is short, lowercase, hyphenated, includes keyword
- [ ] URL has NO dates in it (e.g., `/vivek-bindra-net-worth/` not `/2026/05/vivek-bindra-net-worth/`)

### ✅ Content Structure
- [ ] Only ONE H1 tag — matches article topic, includes keyword
- [ ] H2 subheadings break up content every 200–300 words
- [ ] First paragraph (100 words) mentions keyword naturally
- [ ] Article is minimum 800 words (news), 1500 words (guides)
- [ ] No keyword stuffing — keyword appears naturally, not forced
- [ ] Short paragraphs: maximum 3–4 sentences each
- [ ] Bullet points or numbered lists where appropriate
- [ ] Conclusion paragraph summarises key points

### ✅ Images
- [ ] At least 1 featured image set (minimum 1200×628px)
- [ ] Featured image has descriptive filename: `vivek-bindra-branded-factory-vadodara.jpg`
- [ ] Alt text includes keyword: e.g., `Vivek Bindra at Branded Factory launch Vadodara 2026`
- [ ] Image is compressed (< 200KB) — use TinyPNG or Smush plugin
- [ ] No copyright violations — use free stock (Unsplash, Pexels) or original photos

### ✅ Schema & Technical
- [ ] NewsArticle JSON-LD schema added (auto via plugin or manually)
- [ ] `datePublished` matches actual publish date
- [ ] `author.name` matches the author shown on the page
- [ ] Canonical URL is set to the article's own URL
- [ ] Open Graph tags present (title, description, image)

### ✅ Internal Linking
- [ ] At least 2 links to other articles on vivekbindranews.com
- [ ] At least 1 link from an older article TO this new article (update old articles)
- [ ] Links use descriptive anchor text (NOT "click here")
- [ ] No broken links

### ✅ Author & E-E-A-T
- [ ] Author name is displayed on the article page
- [ ] Author name links to their author bio page
- [ ] Category is correctly assigned
- [ ] Tags added (3–5 relevant tags, e.g., "Vivek Bindra", "Bada Business", "startup")

## Title Tag Formulas (Use These Templates)

**For breaking news:**
```
Vivek Bindra [Event/Action]: [Specific Detail] | VivekBindraNews
```
Example: `Vivek Bindra Inaugurates Branded Factory Vadodara — May 2026 | VivekBindraNews`

**For explainers/guides:**
```
What Is [Topic]? Complete Guide [Year] | Vivek Bindra News
```
Example: `What Is Bada Business? Complete Guide 2026 | Vivek Bindra News`

**For net worth/profile articles:**
```
Vivek Bindra Net Worth [Year]: Career, Companies & Assets | VivekBindraNews
```

**For reviews:**
```
Bada Business [Program] Review [Year]: Is It Worth It? | Vivek Bindra News
```

## Meta Description Templates

**Breaking news:**
```
Vivek Bindra attended the [event] in [location] on [date]. Here's everything that happened and what it means for Indian entrepreneurs.
```

**Explainer:**
```
Bada Business is Dr Vivek Bindra's [one sentence]. Learn about its courses, pricing, and entrepreneur success stories in this complete 2026 guide.
```

**Net worth:**
```
Dr Vivek Bindra's estimated net worth in 2026 is [X]. Explore his companies, YouTube earnings, and Bada Business revenue breakdown.
```
